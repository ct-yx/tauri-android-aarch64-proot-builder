# Security and asset policy

This repository is intentionally limited to build environment automation.

Do not commit:

- third-party web application bundles
- reverse-engineered JavaScript bundles
- proprietary CAD resources
- `models.zip`
- wasm binaries from commercial products
- generated APK/AAB files
- signing keys

Use this repository to reproduce the build environment only. Your own application assets should live in your own private project unless you have permission to redistribute them.
